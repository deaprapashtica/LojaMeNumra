# Loja me Numra
Loja me Numra është një lojë 2D e zhvilluar në Java duke përdorur Swing.

## Si funksionon loja
- Numra random bien nga pjesa e sipërme e ekranit
- Lojtari lëviz platformën majtas-djathtas me tastierë
- Duhet të kapet vetëm numri target
- Kapja e saktë rrit pikët
- Kapja e gabuar ose humbja e target-it ul jetët
- Loja përfundon kur jetët mbarojnë

## Kontrollet
- Left Arrow – lëviz majtas
- Right Arrow – lëviz djathtas
- Enter – restart pas Game Over

## Teknologjitë
- Java
- Swing
- Timer
- Collision detection me Rectangle
